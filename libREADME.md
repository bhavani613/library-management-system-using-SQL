# Library Management System (SQL)

## Description
This project implements a simple Library Management System using SQL. It manages books, library members, and issued book records. The system demonstrates database creation, table relationships, and SQL queries to track book borrowing and analyze library data.

## Features
- Manage books and members
- Track issued and returned books
- Find books currently issued
- Count books by category
- Identify most borrowed books

## Technologies Used
- SQL
- MySQL / SQLite

## Database Tables
1. Books
2. Members
3. Issued_Books

## How to Run
1. Open MySQL Workbench or any SQL environment.
2. Copy the SQL code into a new query tab.
3. Execute the script to create tables and insert data.
4. Run the queries to analyze the library data.

## Example Queries
- View all books
- Find books currently issued
- Count books by category
- Identify most borrowed books

## Learning Outcomes
- Database creation
- Table relationships
- JOIN operations
- Aggregation functions
- Basic data analysis with SQL
